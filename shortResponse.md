## Question 1 — Describe

What is the difference between a static unit like `px` and a responsive unit like `%` or `vh`?

Describe what makes a unit responsive and why that matters when building a website.

The difference between a static unit like px and %/vh is that px never changes it always stays the samme while %/vh adjusts to the devices screen size making it easier for the viewer to see. A responsive unit changes its size/adjusts,this matters when building a website because this then works on all type of devices not just one.
